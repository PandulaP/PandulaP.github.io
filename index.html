Research

Designing a sample-efficient RL Setup with NLP

University of Potsdam — Potsdam, Germany

In this research project, I propose a framework that can be used to integrate the reward shaping process into any dynamical system on which RL methods are used to find optimal policies. In the proposed setup, the RL agent’s learning phase is made sample efficient by creating shaping rewards that are generated according to human-provided natural language instructions. This setup enables human instructors to observe the learning process of an RL agent and give natural language instructions to guide the agent to complete the assigned tasks quickly. Please refer to the paper that I have written about this project for further information.
• Link to project : Sample-efficient RL Setup with NLP - Project folder
• Link to paper : Sample-efficient RL Setup with NLP - Summary Paper
